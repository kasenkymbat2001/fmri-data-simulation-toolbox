# fmri-data-simulation-toolbox
Toolbox for Synthetic fMRI Activity and Connectivity Data

## Project Status

This project is currently under active development. The main structure and core functionality have been implemented, including template loading, ROI definition, experimental design configuration, HRF convolution, synthetic 4D fMRI data generation, and BIDS-compatible export. 

Further improvements are planned, including extended noise modeling, resting-state simulation, improved connectivity tools, and interface optimization.

## Key Features

- Loading and visualization of NIfTI brain templates
- Definition of regions of interest
- Experimental design configuration
- HRF convolution
- ROI-design matrix generation
- Synthetic 4D fMRI data creation
- BIDS-compatible export structure

## Technologies

Python, NumPy, NiBabel, Matplotlib, Tkinter

## Purpose

The toolbox was developed as part of a Master's thesis in Neuroscience to support simulation-based testing of fMRI activity and connectivity analysis pipelines.
